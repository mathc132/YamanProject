Olá, segue algumas instruções para execução do projeto de automação realizado:

- Como o projeto foi desenvolvido com o framework Maven, todo o ambiente já está configurado para a execução.
- Para rodar deve ser executado a raiz do projeto (yaman_project) utlizando o JUnit.
- As features estão no diretorio: /yaman_project/src/test/java/features e como foram implementadas
mais de um tag é possível executar todos os cenarios ou cada um individualmente.
  Para alterar as tags ir na classe Runner no diretorio : /yaman_project/src/test/java/Runner/Runner.java.
